# sql-product-recommendation-system

## Overview

A SQL-based product recommendation and comparison system that helps users identify the best products by comparing prices, ratings, rankings, and available discounts.

# Key Features
* Comparative analysis of products across multiple parameters
* Ranking-based product recommendations
* Price and discount evaluation
* Query-driven product filtering and sorting
* Efficient retrieval of product insights using SQL
*Technologies Used

# Technologies Used
- SQL
- MySQL
- Python
  
# Project Objective

The primary objective of this project is to demonstrate the use of SQL for data retrieval, filtering, aggregation, and analytical decision-making. By leveraging structured queries, the system assists users in selecting products that offer the best overall value based on predefined criteria.
